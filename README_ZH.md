# MeshCombiner
- [English Manual](./README.md)

## 概述
Unity 网格合成器。

## 需求
- Unity 场景搭建时，大量静态的模型（例如：遍地的石块）消耗过多的性能，且在Unity层次面板中不利于管理，希望将这些零散的模型合并成一个模型网格。

## 环境
- Unity 5.0 或更高版本。
- .Net Framework 3.0 或更高版本。

## 实现
- MeshCombiner：绘制扩展编辑器窗口，合并模型网格。
