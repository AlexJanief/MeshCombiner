  [Summeray]
    This tool can be used to combine multi methes to a mesh.
--------------------------------------------------------------------------
  [Environment]
    This tool applies to Unity 5.0, .Net Framework 3.0 or above version.
--------------------------------------------------------------------------
  [Usage]
    Find the menu item "Tool/Mesh Combiner" in Unity editor menu bar
    and click it or press combination key Alt+M to open the editor window.
    Create an empty gameobject as "Meshes Root", drag the target meshe
    gameobjects to under the "Meshes Root".
    Create an empty gameobject as "Mesh Save".
    Click the "Combine" button to open the save new mesh dialog.
    Select a path and input a file name to the dialog and save.
